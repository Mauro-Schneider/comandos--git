git init = iniciar git  -  iniciar repositorio
git clone = clonar repositorio 
git add = añadir archivo al area de preparacion 
git commit -m "mensaje del comit" = ejecutar un commit
git log = ver el historial de commits
git diff commit_id commit_id (primeros sinco caracteres de los commits a comparar) ver diferencias entre los archivos de los dos commits.
git checkout comit_id = volver a la vercion del commit indicada
git tag -a  nombre_de_la_etiqueta -m "mensaje explicativo"
git branch nombre_de_la_rama (crear una rama)
git merge rama (unir rama actual con la rama especificada)
